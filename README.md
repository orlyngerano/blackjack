# Blackjack

Terminal based game.

## Usage

### Starts the game

```bash
% ./BlackJackGame
```

## Installing Build Tools

Follow CMake installation [Guide](https://cmake.org/install/)

## Building

```bash
cmake -S . -B build
cd build
make
```

The built file is located in root directory **build**.

## Testing

```bash
./BlackJackGame
```
